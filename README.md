MediLab is a full-stack medical website built with **Laravel**, featuring a modern **Admin Dashboard** and a public **Frontend Website**.  
The project demonstrates clean architecture, role-based authentication, and full CRUD functionality.

---

## Features

### Admin Panel
- Separate **Admin Authentication (Custom Guard)**
- Secure Admin Login Page
- Dashboard with statistics UI
- CRUD Management:
  - Departments
  - Services
  - Doctors
  - FAQs
  - Testimonials
  - Gallery
  - Appointments
- Image Upload & Preview
- Pagination & Validation

### Frontend Website
- Home Page
- About
- Services
- Doctors
- Gallery
- FAQ
- Contact
- Responsive Design

- **Laravel 12**
- **PHP 8.2**
